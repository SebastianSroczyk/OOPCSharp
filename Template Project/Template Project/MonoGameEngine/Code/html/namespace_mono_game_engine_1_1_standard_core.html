enum  LoopType { Standard , Bounce , None }
 Represents the style of loop that is desired for an animation.
- Standard loops the animation from the start to the end, before starting from the beginning again.
- Bounce bounces the animation loop forwards and backwards when it runs out of frames in its current direction.
- None will only play the animation once before stopping on the final frame.
 
enum  BackgroundType {
  Stretch , Wrap , HorizontalScroll , VerticalScroll ,
  FullScroll
}
 Enum used to control Background rendering behaviour.
- Stretch will fill the current window with the held image.
- Wrap will tile the background image across the current window without stretching. Does not allow scrolling.
- HorizontalScroll wraps the background image across the current window, but only allows scrolling horizontally.
- VerticalScroll wraps the background image across the current window, but only allows scrolling vertically.
- FullScroll wraps the background image across the current window, and allows scrolling in all directions.
