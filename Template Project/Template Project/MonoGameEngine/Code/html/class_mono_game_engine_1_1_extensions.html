Draws a circle at a given origin with the requested radius. The circle will look smoother depending on how many points are requested.

Parameters
spriteBatchThe SpriteBatch currently being used to batch for rendering.
originThe center point that the circle should use for rendering.
radiusThe radius of the desired circle to be rendered.
points[Optional] The number of points that make up the circle. 24 by default. The minimum acceptable number of points is 3.
thickness[Optional] The thickness of circle outline. 1.0f by default.
colour[Optional] The desired rendering colour.
