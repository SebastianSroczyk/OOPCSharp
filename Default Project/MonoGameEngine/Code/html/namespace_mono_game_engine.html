enum  SFXOverlapRule { Ignore , Overwrite , Overlay }
 Controls how sound effects are handled when a request to play an existing sound is made.
- Ignore discards any extra sfx of the same name.
- Overwrite replaces the current instance of the sfx.
- Overlay allows multiples of a single sfx to play at once.
 
enum  MouseButton { Left , Right , Middle }
 Enum representing the three main mouse buttons. Used when handling mouse click inputs.
- Left represents a left mouse button click.
- Right represents a right mouse button click.
- Middle represents a middle mouse button click.
 
enum  TransitionType {
  Fade , Fill , SwipeLeft , SwipeRight ,
  SwipeUp , SwipeDown
}
 Enum used to represent the animation style desired for a transition.
- Fade will perform a fullscreen fade in to/out from a solid colour by adjusting the transparency level.
- Fill will perform a scaling box fill in/out using a solid colour.
- SwipeLeft will perform a swipe transition in a leftward motion, using a single colour panel.
- SwipeRight will perform a swipe transition in a rightward motion, using a single colour panel.
- SwipeUp will perform a swipe transition in a upward motion, using a single colour panel.
- SwipeDown will perform a swipe transition in a downward motion, using a single colour panel.
